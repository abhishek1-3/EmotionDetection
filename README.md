## 🧠 About the Project

This system uses data augmentation and a CNN architecture to improve generalization and accuracy on unseen data. The model processes 48x48 grayscale facial images and outputs one of seven emotion classes.

## 🚀 Features
Image preprocessing using ImageDataGenerator

Data augmentation (rotation, zoom, shear, flip)

CNN model with multiple convolutional + pooling layers

Dropout layers to prevent overfitting

Trained on categorized dataset (train/ and test/)

Saves trained model (.h5) for later use

Supports real-time prediction (via webcam script)

Python 🐍

Keras / TensorFlow

OpenCV
